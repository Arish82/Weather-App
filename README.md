# Weather-App
## Clone project in your locals
    git clone git@github.com:Arish82/Weather-App.git<br>
    cd /'Weather App'

## Now download dependencies
    npm i express<br>
    npm i requests<br>
    npm i hbs
  
## Now run the app
    cd src
    node index.js
  
## Now open your browser
    localhost:8000
